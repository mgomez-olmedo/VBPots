VariableEliminationTreeStats 

Companion object VariableEliminationTreeStats

class VariableEliminationTreeStats extends VariableElimination

Class for storing information about the execution of Variable Elimination algorithm using tree and another representations of interest. This allows to perform the propagation using trees and then, once a potential is computed, is converted into another representations for getting insight about potential sizes produced during inference
