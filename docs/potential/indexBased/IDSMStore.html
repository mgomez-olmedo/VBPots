IDSMStore 

Companion object IDSMStore

case class IDSMStore(variables: VariableSet, indexesSet: ArrayBuffer[Set[Long]], values: ArrayBuffer[Double]) extends ValueDrivenStore with Combiner with Marginalizer with Product with Serializable

Class for storing values with the following structure: values are stored in an array; each value is paired with the same position in another array where indices are stored in sets. In this case mutable arrays are employed

variables

domain of the related potential

indexesSet
